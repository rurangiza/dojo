## Reminders in Python

#### Converting list to dictionnary counter
```python
from collections import Counter

array = [1, 2, 3, 4, 2, 2, 1, 4]
count = Counter(arr)
# Counter({2: 3, 1: 2, 4: 2, 3: 1})
```

#### Walrus operator
```python
import time

def do_something() -> int:
    sleep(1)
    return 42

# slow
if do_something() == 42:
    return do_something()

# much better
x = do_something()
if x == 42:
    return x

# more concise, better? maybe
if (x := do_something()) == 42:
    return x
```
